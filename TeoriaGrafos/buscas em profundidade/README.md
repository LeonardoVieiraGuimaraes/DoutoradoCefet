# Buscas em Grafos

Este módulo agrega materiais de travessia em grafos. Atualmente inclui:

- Busca em Largura (BFS)
  - [01_busca_em_largura_bfs.ipynb](../buscas%20em%20largura/01_busca_em_largura_bfs.ipynb)
  - [02_exercicios_bfs.ipynb](../buscas%20em%20largura/02_exercicios_bfs.ipynb)
- Busca em Profundidade (DFS)
  - [03_busca_em_profundidade_dfs.ipynb](./03_busca_em_profundidade_dfs.ipynb)

## Pré-requisitos
- Python 3.10+ gerenciado com `uv`
- Pacotes: `networkx`, `matplotlib`, `numpy`, `jupyter`

## Como usar
1. Abra os notebooks acima no VS Code/Jupyter e execute as células sequencialmente.
2. Ajuste os grafos/exemplos para experimentar diferentes cenários.

## Referências
- Cormen, Leiserson, Rivest, Stein — Introduction to Algorithms (Cap. 22)
- Sedgewick & Wayne — Algorithms
- Bondy & Murty — Graph Theory
