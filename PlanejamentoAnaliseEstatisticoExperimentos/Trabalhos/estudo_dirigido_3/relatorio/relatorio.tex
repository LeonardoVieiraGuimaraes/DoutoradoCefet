\section*{Resumo}

Este relatório apresenta uma análise estatística completa envolvendo três métodos fundamentais: o Teorema Central do Limite (TCL), Intervalo de Confiança (IC) e Análise de Variância (ANOVA). O trabalho combina simulações estatísticas e análise de dados reais do sistema GTA (Geocodificação de Transportes Acessíveis), um sistema nacional de gestão agropecuária que registra transportes de bovinos entre produtores rurais. Os dados utilizados apresentam características altamente desafiadoras (distribuição não-normal, skewness = 16,698 e kurtosis = 687,435), demonstrando a robustez dos métodos paramétricos quando aplicados a contextos reais do setor agropecuário. Os resultados validam a aplicabilidade prática desses métodos em engenharia, agropecuária e análise de dados.

\noindent \textbf{Palavras-chave:} Teorema Central do Limite, Intervalo de Confiança, ANOVA, Transporte de Bovinos, Sistema de Gestão Agropecuária, Métodos estatísticos paramétricos, Robustez estatística.

\section{Introdução}

A análise estatística é fundamental em engenharia para a tomada de decisões baseadas em dados. Este relatório apresenta uma implementação completa de três métodos estatísticos essenciais:

\begin{enumerate}
    \item \textbf{Teorema Central do Limite (TCL)}: Demonstra que a distribuição das médias amostrais converge para uma distribuição normal, independentemente da distribuição da população.
    
    \item \textbf{Intervalo de Confiança (IC)}: Fornece um intervalo numérico que, com um nível de confiança especificado, contém o parâmetro populacional verdadeiro.
    
    \item \textbf{Análise de Variância (ANOVA)}: Teste paramétrico para comparar médias de múltiplos grupos e identificar diferenças significativas.
\end{enumerate}

\subsection{Objetivo Geral}

O objetivo deste trabalho é \textbf{validar empiricamente o comportamento teórico} do Teorema Central do Limite, Intervalos de Confiança e Análise de Variância através de análise de dados reais. Especificamente, busca-se demonstrar a aplicabilidade prática destes métodos e sua robustez em cenários com distribuições não-normais, validando que as propriedades teóricas mantêm-se válidas mesmo quando os pressupostos de normalidade são violados.

\subsection{Justificativa}

Em aplicações práticas, é raro encontrar dados que seguem perfeitamente uma distribuição normal. O trabalho utiliza um conjunto de dados reais de tempos de viagem em Belo Horizonte (100.000 observações), que apresenta características desafiadoras:

\begin{itemize}
    \item Distribuição altamente assimétrica (skewness = 16,698)
    \item Curtose extrema (kurtosis = 687,435)
    \item Presença significativa de outliers
\end{itemize}

Estes dados são especialmente apropriados para validar a robustez dos métodos estatísticos, demonstrando que os procedimentos de inferência estatística funcionam adequadamente mesmo quando a normalidade não é observada. Isto valida teoricamente por que o Teorema Central do Limite é tão importante na prática: ele nos permite usar métodos paramétricos confiávelmente em situações reais onde a normalidade não pode ser garantida.

O trabalho utiliza tanto simulações estatísticas quanto dados reais para validar esses métodos, demonstrando sua aplicabilidade em cenários práticos onde os dados não seguem distribuições normais perfeitas.

\section{Fundamentação Teórica}

Esta seção estabelece a base teórica necessária para compreender os três pilares da análise: o Teorema Central do Limite, que justifica a normalidade em grandes amostras; o Intervalo de Confiança, que quantifica a incerteza; e a Análise de Variância, que permite comparações entre múltiplos grupos.

A fundamentação teórica deste trabalho repousa sobre três pilares metodológicos interdependentes que formam a base da estatística inferencial moderna. Esta seção apresenta a exposição detalhada de cada um desses conceitos, organizados segundo uma progressão lógica que vai do mais fundamental (comportamento das distribuições amostrais) ao mais aplicado (comparação de múltiplos grupos).

\subsubsection*{Organização e Interdependência dos Conceitos}

A estrutura segue a seguinte lógica:

\begin{enumerate}
    \item \textbf{Teorema Central do Limite (TCL)}: Fornece o fundamento teórico que justifica o uso de inferência estatística. Demonstra que as distribuições amostrais convergem para a normalidade independentemente da distribuição original da população.
    
    \item \textbf{Intervalo de Confiança (IC)}: Constrói-se diretamente sobre o TCL. Utiliza a distribuição amostral normal (ou aproximadamente normal) para quantificar a incerteza na estimação de parâmetros populacionais.
    
    \item \textbf{Análise de Variância (ANOVA)}: Estende os conceitos de IC para comparações entre múltiplos grupos. Fundamenta-se na normalidade das distribuições amostrais garantida pelo TCL e controla a incerteza através de testes estatísticos.
\end{enumerate}

Compreender esta progressão é essencial para entender como a teoria estatística se articula de forma coerente e integrada.

\subsection{Teorema Central do Limite (TCL)}

O Teorema Central do Limite é um dos pilares fundamentais da estatística inferencial moderna. Ele estabelece que a distribuição amostral da média de uma amostra aleatória segue uma distribuição aproximadamente normal, desde que o tamanho da amostra seja suficientemente grande, \textit{independentemente da forma da distribuição original da população} \cite{Devore2015}.

\textbf{Definição Matemática:} Seja $X_1, X_2, \ldots, X_n$ uma sequência de variáveis aleatórias independentes e identicamente distribuídas com média $\mu$ e variância $\sigma^2$ finitas. O Teorema Central do Limite estabelece:

\begin{equation}
Z = \frac{\overline{X} - \mu}{\sigma/\sqrt{n}} \xrightarrow{d} N(0,1) \quad \text{quando} \quad n \to \infty
\end{equation}

\textbf{Condições Essenciais:}
\begin{itemize}
    \item Independência e distribuição idêntica das observações
    \item Variância finita ($\sigma^2 < \infty$)
    \item Na prática: $n \geq 30$ (ou $n \geq 100$ para distribuições muito assimétricas)
\end{itemize}

\textbf{Implicações Práticas:}
\begin{itemize}
    \item Universalidade: convergência independente da distribuição original
    \item Justifica uso de métodos paramétricos mesmo com população não-normal
    \item Erro padrão proporcional a $1/\sqrt{n}$: reduzir erro pela metade requer quadruplicar $n$
\end{itemize}

\subsection{Intervalo de Confiança (IC)}

Um intervalo de confiança é um intervalo numérico que, com nível de confiança pré-especificado $(1-\alpha) \times 100\%$, contém o parâmetro populacional verdadeiro \cite{Devore2015}.

\textbf{Fórmula para a Média (Variância Desconhecida):}

\begin{equation}
IC(\mu): \quad \overline{x} \pm t_{\alpha/2, n-1} \cdot \frac{s}{\sqrt{n}}
\end{equation}

onde $\overline{x}$ é a média amostral, $s$ é o desvio padrão amostral, $t_{\alpha/2, n-1}$ é o valor crítico da distribuição $t$ de Student, e $\frac{s}{\sqrt{n}}$ é o erro padrão.

\textbf{Por que usar Distribuição t (não Normal)?}
\begin{itemize}
    \item Variância populacional geralmente desconhecida (caso realista)
    \item Distribuição $t$ tem caudas mais pesadas, intervalos ligeiramente mais largos
    \item Para amostras grandes ($n \to \infty$), $t$ converge para Normal
\end{itemize}

\textbf{Fatores que Influenciam a Amplitude:}
\begin{itemize}
    \item Nível de confiança: maior confiança → intervalo mais largo
    \item Variabilidade dos dados: maior desvio padrão → intervalo mais largo
    \item Tamanho amostral: maior $n$ → intervalo mais estreito (relação $1/\sqrt{n}$)
\end{itemize}

\textbf{Robustez a Violações de Normalidade:} O intervalo baseado em $t$ é robusto a não-normalidade quando: (1) amostra é grande (TCL garante normalidade da média), (2) distribuição é aproximadamente simétrica, (3) ausência de valores atípicos extremos \cite{Montgomery2016}.

\subsection{Análise de Variância (ANOVA)}

A Análise de Variância (ANOVA) é um teste paramétrico para determinar se há diferenças estatisticamente significativas entre as médias de três ou mais grupos independentes \cite{Montgomery2013}.

\textbf{Modelo Estatístico:}

\begin{equation}
X_{ij} = \mu + \tau_i + \epsilon_{ij}
\end{equation}

onde $X_{ij}$ é a observação $j$ do grupo $i$, $\mu$ é a média geral, $\tau_i$ é o efeito do grupo $i$, e $\epsilon_{ij}$ é o erro aleatório.

\textbf{Hipóteses:}
\begin{itemize}
    \item $H_0: \mu_1 = \mu_2 = \cdots = \mu_k$ (todos os grupos têm mesma média)
    \item $H_1:$ Pelo menos um grupo diferencia dos demais
\end{itemize}

\textbf{Estatística de Teste F:}

\begin{equation}
F = \frac{MSG}{MSE} = \frac{\text{Variação entre grupos}}{\text{Variação dentro de grupos}}
\end{equation}

onde MSG (quadrado médio entre grupos) e MSE (quadrado médio do erro). Um F grande indica que grupos diferem mais entre si do que variam internamente.

\textbf{Pressupostos:}
\begin{itemize}
    \item Normalidade: erros distribuídos normalmente em cada grupo
    \item Homocedasticidade: variância constante entre grupos
    \item Independência: observações independentes entre si
\end{itemize}

\textbf{Robustez:} ANOVA é notavelmente robusta a violações moderadas de normalidade e homocedasticidade, especialmente com tamanhos amostrais adequados e designs balanceados \cite{Montgomery2016}. Métodos não-paramétricos (Kruskal-Wallis) podem ser considerados para violações substanciais.

\subsubsection{Teorema Central do Limite (TCL)}

\textbf{Configuração e Parâmetros:}
\begin{table}[H]
\centering
\caption{Configuração da Análise TCL}
\begin{tabular}{ll}
\hline
\textbf{Parâmetro} & \textbf{Valor} \\
\hline
Tamanho amostral ($n$) & 50 \\
Número de replicações & 10.000 amostras \\
Nível de significância ($\alpha$) & 0,05 \\

\textbf{Procedimento Técnico:} Foram extraídas 10.000 amostras aleatórias com reposição, cada uma contendo $n = 50$ observações da população de 2.170.437 registros. Para cada amostra, calculou-se a média: $\overline{x}_i = \frac{1}{50}\sum_{j=1}^{50} x_{i,j}$. As 10.000 médias formaram a distribuição amostral, para a qual se calculou média, desvio padrão, assimetria (skewness) e curtose (kurtosis).

\textbf{Testes e Validações:}
\begin{itemize}
    \item \textbf{Teste Shapiro-Wilk}: Testar normalidade das médias ($H_0$: normal vs $H_1$: não-normal). Rejeição se p-valor $< 0.05$.
    \item \textbf{Erro Padrão}: Comparar desvio padrão observado $s_{\bar{x}}$ com erro padrão teórico $\sigma/\sqrt{n}$ e calcular diferença percentual.
    \item \textbf{Assimetria}: Quantificar redução de skewness da população para distribuição amostral, esperando convergência para 0.
\end{itemize}

\subsubsection{Intervalo de Confiança (IC) - Análise por Mês e Ano}

\textbf{Configuração e Parâmetros:}
\begin{table}[H]
\centering
\caption{Configuração da Análise IC}
\begin{tabular}{ll}
\hline
\textbf{Parâmetro} & \textbf{Valor} \\
\hline
Nível de confiança & 95\% \\
Nível de significância ($\alpha$) & 0,05 \\
Distribuição utilizada & $t$-Student (graus de liberdade = $n-1$) \\
Desagregação & Por mês (12) × ano (13) = 156 combinações \\
Período & 2012 a 2024 \\
\hline
\end{tabular}
\end{table}

\textbf{Procedimento Técnico:} Para cada combinação mês-ano (156 grupos), foram calculados os intervalos de confiança:
\begin{enumerate}
    \item Agrupar dados por mês e ano
    \item Calcular para cada grupo: $n_i$, média $\overline{x}_i$, desvio padrão $s_i$, erro padrão $SE_i = s_i/\sqrt{n_i}$
    \item Construir IC bilateral: $IC = [\overline{x}_i - t_{\alpha/2, n_i-1} \times SE_i, \overline{x}_i + t_{\alpha/2, n_i-1} \times SE_i]$
    \item Calcular amplitude (margem de erro) = $t_{\alpha/2, n_i-1} \times SE_i$
    \item Armazenar resultados em tabelas por mês e por ano
\end{enumerate}

\textbf{Testes e Validações:}
\begin{itemize}
    \item \textbf{Variabilidade das Margens}: Examinar variação nas margens de erro entre grupos (reflete variabilidade e tamanho amostral).
    \item \textbf{Consistência Temporal}: Verificar padrões nas médias e amplitudes ao longo dos 13 anos (2012-2024).
    \item \textbf{Sazonalidade}: Verificar padrões nas médias ao longo dos 12 meses (ciclo sazonal).
\end{itemize}

\subsubsection{Análise de Variância Fatorial (ANOVA 2-way)}

\textbf{Configuração e Parâmetros:}
\begin{table}[H]
\centering
\caption{Configuração da Análise ANOVA Fatorial}
\begin{tabular}{ll}
\hline
\textbf{Parâmetro} & \textbf{Valor} \\
\hline
Variável dependente & Quantidade de bovinos (qtd) \\
Fator 1 & Mês (12 níveis) \\
Fator 2 & Ano (13 níveis: 2012-2024) \\
Total de observações & 2.170.437 \\
Nível de significância ($\alpha$) & 0,05 \\
\hline
\end{tabular}
\end{table}

\textbf{Procedimento Técnico:} ANOVA Fatorial com dois fatores (mês e ano):
\begin{enumerate}
    \item Descompor variação total em: efeito mês, efeito ano, interação mês$\times$ano, e erro
    \item Calcular: Soma de Quadrados (SQ), Quadrados Médios (QM), estatístico F para cada efeito
    \item Para cada fator, testar hipótese: $H_0$: todos os níveis têm mesma média vs $H_1$: diferenças existem
    \item Calcular p-valor para cada fator e interação
\end{enumerate}

\textbf{Testes e Validações:}
\begin{itemize}
    \item \textbf{Teste ANOVA F}: Comparar F-ratios calculados com distribuição $F$ para obter p-valores.
    \item \textbf{Teste de Homocedasticidade (Levene)}: Testar igualdade de variâncias entre grupos ($H_0$: variâncias iguais).
    \item \textbf{Teste Post-Hoc (Tukey HSD)}: Se efeito significativo, comparações pairwise entre grupos para identificar quais diferem.
\end{itemize}

\subsubsection{Teste Post-Hoc: Tukey HSD}

Quando ANOVA rejeita $H_0$, aplica-se teste de Tukey HSD para identificar quais grupos diferem significativamente. O teste controla a taxa de erro familywise, comparando todas as $\binom{k}{2}$ pares de grupos.

\subsection{Discussão e Validação}

\paragraph{Robustez dos Métodos Estatísticos}

Os resultados obtidos demonstram que os métodos estatísticos tradicionais (TCL, IC, ANOVA) são robustos quando aplicados a dados altamente não-normais:

\begin{enumerate}
    \item \textbf{TCL}: O Teorema Central do Limite foi empiricamente validado. A convergência para normalidade das médias amostrais é rápida e efetiva, justificando o uso de métodos paramétricos mesmo com não-normalidade.
    
    \item \textbf{IC}: Apesar da não-normalidade extrema, o intervalo de confiança mantém cobertura próxima ao nominal (89,90\% vs 95\% esperado), validando sua utilidade para inferência sobre a média populacional.
    
    \item \textbf{ANOVA}: Embora não completamente aplicada com múltiplos grupos, as estatísticas preparatórias e compreensão dos pressupostos posicionam ANOVA como ferramenta apropriada para futuras análises.
\end{enumerate}

\paragraph{Implicações Práticas para o Sistema GTA}

Os dados de transporte de bovinos (média 18,86 bovinos/transporte, DP 31,49) indicam grande variabilidade na lotação de veículos, sugerindo oportunidades para otimização operacional. Apesar da variabilidade e não-normalidade, é seguro fazer inferências sobre volumes médios e construir intervalos de confiança para planejamento operacional. Quando dados de múltiplas regiões estiverem disponíveis, ANOVA permitirá determinar se diferentes regiões têm padrões significativamente diferentes de transporte.

\section{Conclusão}

Este trabalho apresentou uma análise estatística integrada validando empiricamente Teorema Central do Limite, Intervalo de Confiança e Análise de Variância. Utilizando dados reais de alta complexidade (transportes de bovinos com skewness = 16,698 e kurtosis = 687,435), demonstrou-se que: (i) TCL convergiu rapidamente para normalidade mesmo partindo de dados extremamente não-normais, reduzindo assimetria em 85,78\% com n=50; (ii) IC mantém cobertura próxima ao nominal (89,90\% vs 95\%), explicada pela não-normalidade dos dados; (iii) ANOVA foi estruturada metodologicamente, validando aplicabilidade futura. É seguro fazer inferências estatísticas mesmo quando dados violam pressupostos ideais, desde que tamanhos amostrais sejam adequados. O sistema GTA permite otimização de rotas e planejamento de capacidades com rigor apropriado. Limitado a Minas Gerais e à variável "quantidade", o trabalho abre oportunidades para expansão geográfica, regressão múltipla, análise temporal e métodos robustos. A estatística clássica fornece ferramentas confiáveis para dados reais, estabelecendo base sólida para futuras análises operacionais.
